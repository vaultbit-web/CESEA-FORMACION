Cómo usar estos mockups para material comercial

Cómo se han generado: capturas directas del producto desplegado vía Chrome headless al puerto 8765 (_screenshot.mjs en la raíz). Cada imagen es una sesión real con datos demo precargados — no son renders ni simulaciones.

Para capturar PNG de este mockup: abre este archivo en Chrome, clic derecho → Capturar captura de página completa. O usa la extensión GoFullPage.

Para capturar una sección concreta: clic derecho sobre el mockup → Inspeccionar → selecciona el <div class="scene"> → clic derecho en el panel → Capture node screenshot.

Para regenerar las capturas: ejecuta python -m http.server 8765 en la raíz del proyecto y luego node _screenshot.mjs. Se regenerarán automáticamente las 6 capturas.

Exportar a PDF (para adjuntar a propuestas): Ctrl+P → Guardar como PDF. Elige A4 o Letter horizontal.
